### Running Locally

**Cloning**

```
git clone http://github.com/sqwyer/epm-t3.git
cd epm-t3
```

> **Note:** You must create and fill out a `.env` file, you can use the `.env-example` file as a template.

**Running**

```
npm install
npm run dev
```
